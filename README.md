# ImageMatrix (影像矩阵)

**ImageMatrix** 是一款基于 Python 和 PyQt6 开发的高效桌面图像处理工具。它集成了**智能拼图**与**智能切图**两大核心功能，专为设计师、电商运营和内容创作者打造。

## 核心功能

### 1. 智能拼图 (Merge)
将多张图片自动拼接成一张或多张长图，完美适配电商详情页或长图文排版。
*   **拖拽排序**：支持直接拖拽图片文件，自动按名称排序。
*   **灵活输出**：支持**原图宽度**、**750px**、**1080px** 或 **自定义宽度**导出。
*   **分组拼接**：可设置将所有图片拼成 1 张长图，或自动分割成 N 张（便于文件过大时分段）。

### 2. 智能切图 (Slice) [NEW]
将一张超长图片垂直裁切成多张小图，便于网页加载或平台上传。
*   **智能自动吸附 (Smart Snap)**：独创的内容感知算法，**自动避开文字、产品和复杂图案**，优先在背景空白处进行裁切，不再担心“切断文字”。
*   **自定义切分**：自由拖动滑块选择切分数量（1-50份）。
*   **尺寸调整**：支持在切图的同时调整图片宽度。

## 快速开始

### 运行程序
双击根目录下的脚本即可启动：
`启动ImageMatrix工具.bat`

### 使用指南

#### 拼图模式
1. 切换到 **“拼图 (Merge)”** 标签页。
2. 将图片文件 (JPG/PNG/PSD) 拖入虚线框。
3. 选择**导出宽度**和**拼接数量**。
4. 点击 **“开始拼接并保存到桌面”**。

#### 切图模式
1. 切换到 **“切图 (Slice)”** 标签页。
2. 将长图拖入虚线框。
3. 设置**切分数量**。
4. 勾选 **“智能自动吸附”** (推荐，避免切坏内容)。
5. 点击 **“开始切图并保存到桌面”**。
